# Work Control Board 사용자 매뉴얼

> **Work Control Board(WCB)** 는 제조사 문의, 충전기 연동 검증, 고객 민원 대응 등 업무를 로컬 브라우저에서 관리하는 개인 업무 관제판입니다.
> 모든 데이터는 브라우저 localStorage에 저장되며, 인터넷 연결 없이 사용할 수 있습니다.

## 1. 화면 구성

앱은 **좌측 사이드바 + 메인 컨텐츠** 구조로 되어 있습니다.

| 메뉴 | 경로 | 설명 |
|------|------|------|
| 대시보드 | `/` | 전체 업무 현황 요약 |
| 업무 목록 | `/tasks` | 전체 업무 검색 및 관리 |
| 칸반 | `/board` | 상태별 칸반 보드 |
| 프로젝트 | `/projects` | 프로젝트 관리 |
| 회신 대기 | `/waiting` | 외부 회신 대기 업무 전용 뷰 |
| 설정 | `/settings` | 데이터 백업 / 복원 / 초기화 |

---

## 2. 대시보드

앱을 열면 처음 표시되는 화면입니다. 현재 업무 상태를 한눈에 파악할 수 있습니다.

### 요약 카드

화면 상단에 6개의 지표 카드가 표시됩니다.

| 카드 | 설명 |
|------|------|
| 오늘 할 일 | 오늘 마감이거나 오늘로 시작한 업무 수 |
| 지연 업무 | 마감일이 지났거나 상태가 '지연'인 업무 수 (건수가 있으면 강조 표시) |
| 회신 대기 | 상태가 '회신 대기'인 업무 수 |
| 이번 주 마감 | 이번 주 안에 마감인 업무 수 |
| 이번 주 완료 | 이번 주에 완료 처리한 업무 수 |
| 진행 중 프로젝트 | 상태가 '진행 중' 또는 '예정'인 프로젝트 수 |

### 업무 섹션

요약 카드 아래에 4개의 업무 목록이 표시됩니다.

- **오늘 할 일** — 오늘 처리가 필요한 업무
- **지연 업무** — 처리가 늦어진 업무
- **회신 대기** — 외부 회신을 기다리는 업무
- **최근 업무** — 최근 수정된 미완료 업무

각 섹션에는 최대 4건이 표시됩니다. **전체 보기 →** 링크를 클릭하면 해당 필터가 적용된 업무 목록으로 이동합니다.

### 빠른 업무 추가

대시보드 중앙에 있는 빠른 추가 폼으로 제목, 우선순위, 마감일, 다음 액션을 입력해 즉시 업무를 등록할 수 있습니다. 상태는 자동으로 **신규**로 설정됩니다.

## 3. 업무 목록

전체 업무를 검색하고 필터링할 수 있는 메인 관리 화면입니다.

### 빠른 필터

버튼 형태로 자주 쓰는 필터를 바로 적용할 수 있습니다. 해당하는 업무 건수가 배지로 표시됩니다.

| 버튼 | 필터 기준 |
|------|----------|
| 전체 | 모든 업무 |
| 오늘 할 일 | 오늘 마감이거나 오늘로 시작한 업무 |
| 이번 주 마감 | 이번 주 안에 마감인 업무 |
| 지연 업무 | 마감일 초과 또는 상태가 '지연' |
| 회신 대기 | 상태가 '회신 대기' |
| 긴급 업무 | 우선순위가 '긴급' (완료·취소 제외) |
| 완료 숨기기 | 완료·취소 업무를 목록에서 숨김 (토글) |

### 검색 및 상세 필터

빠른 필터 아래에 있는 검색창과 드롭다운으로 복합 조건을 설정할 수 있습니다.

- **키워드 검색** — 업무 제목, 담당자 등 텍스트 검색
- **상태 필터** — 8개 상태 중 하나 선택
- **우선순위 필터** — 4개 우선순위 중 하나 선택
- **카테고리 필터** — 10개 카테고리 중 하나 선택
- **프로젝트 필터** — 등록된 프로젝트 중 하나 선택

여러 필터를 동시에 적용할 수 있습니다. **초기화** 버튼으로 모든 필터를 한 번에 지웁니다.

### 업무 테이블

필터 결과가 테이블로 표시됩니다.

| 컬럼 | 설명 |
|------|------|
| 제목 | 업무명 (클릭 시 상세 보기) |
| 상태 | 현재 상태 배지 |
| 우선순위 | 우선순위 배지 |
| 카테고리 | 업무 분류 |
| 프로젝트 | 연결된 프로젝트명 |
| 담당자 | 담당자 또는 협력사 |
| 다음 액션 | 다음에 해야 할 일 |
| 마감일 | 마감일 (초과 시 빨간색 강조) |
| 생성일 | 업무 생성 날짜 |
| ⋯ | 상태 변경 / 수정 / 삭제 메뉴 |

행을 클릭하면 **업무 상세 다이얼로그**가 열립니다.

## 4. 업무 추가 및 수정

### 업무 추가

**업무 목록** 페이지 우상단의 **업무 추가** 버튼 또는 대시보드의 **빠른 추가 폼**을 사용합니다.

### 업무 필드 전체 목록

업무 추가/수정 다이얼로그에서 입력할 수 있는 필드입니다.

| 필드 | 필수 | 설명 |
|------|:----:|------|
| 제목 | ✓ | 업무명 |
| 상태 | | 현재 업무 상태 (기본값: 신규) |
| 우선순위 | | 긴급 / 높음 / 보통 / 낮음 (기본값: 보통) |
| 카테고리 | | 업무 유형 분류 (기본값: 기타) |
| 프로젝트 | | 연결할 프로젝트 |
| 담당자 / 협력사 | | 담당자 이름 또는 외부 업체명 |
| 충전기 모델 | | 관련 충전기 모델명 |
| 오류 코드 | | 관련 오류 코드 |
| 시작일 | | 업무 시작 예정일 |
| 마감일 | | 업무 완료 기한 |
| 요청일 | | 외부에 요청한 날짜 (회신 대기 업무용) |
| 팔로업 예정일 | | 회신이 없을 경우 다시 연락할 날짜 |
| 설명 | | 업무 상세 내용 |
| 다음 액션 | | 현재 시점에서 해야 할 다음 행동 |
| 관련 링크 | | 참고 URL |
| 메모 | | 기타 메모 |

> **팁**: 회신 대기 업무는 **요청일**과 **팔로업 예정일**을 입력해두면 회신 대기함에서 팔로업 알림을 확인할 수 있습니다.

### 업무 복제

업무 상세 화면의 **복제** 버튼을 사용하면 해당 업무의 모든 내용을 복사한 새 업무가 생성됩니다. 복제된 업무는 제목 앞에 `[복사]`가 붙고 상태가 **신규**로 초기화됩니다.

## 5. 업무 상세 보기

업무 행 또는 카드를 클릭하면 상세 다이얼로그가 열립니다.

### 상단 액션 버튼

완료·취소 상태가 아닌 업무에서 아래 액션을 바로 실행할 수 있습니다.

| 버튼 | 동작 |
|------|------|
| 완료 | 상태를 '완료'로 변경하고 완료 시각 기록 |
| 진행 중 | 상태를 '진행 중'으로 변경 |
| 회신 대기 | 상태를 '회신 대기'로 변경 |
| 지연 | 상태를 '지연'으로 변경 |
| 복제 | 업무 복사본 생성 |
| 수정 | 업무 편집 다이얼로그 열기 |
| 삭제 | 업무 삭제 (확인 후 삭제) |

### 활동 로그

업무 하단에 최근 10건의 활동 이력이 표시됩니다.

- 업무 생성
- 필드 수정
- 상태 변경 (예: 신규 → 진행 중)
- 완료 처리
- 삭제

## 6. 칸반 보드

업무를 상태별 컬럼으로 시각화하는 뷰입니다.

### 컬럼 구성

| 컬럼 | 상태 |
|------|------|
| 신규 | new |
| 진행 중 | in_progress |
| 회신 대기 | waiting |
| 검토 필요 | review |
| 보류 | hold |
| 지연 | delayed |
| 완료 | done |
| 취소 | cancelled |

완료·취소 컬럼은 흐릿하게 표시되어 완료된 업무와 진행 중인 업무를 구분합니다. 각 컬럼 헤더에는 현재 건수가 표시됩니다.

### 업무 카드

카드를 클릭하면 업무 상세 다이얼로그가 열립니다. 카드 우상단의 `⋯` 메뉴에서 상태 변경, 수정, 삭제를 할 수 있습니다.

## 7. 프로젝트 관리

여러 업무를 묶어 프로젝트 단위로 진행 상황을 관리합니다.

### 프로젝트 카드

각 프로젝트는 카드 형태로 표시되며 다음 정보를 보여줍니다.

- **프로젝트명 및 상태 배지**
- **진행률 바** — 완료 업무 / 전체 업무 비율
- **업무 통계** — 완료 / 전체, 지연 건수, 회신 대기 건수
- **목표 완료일** — 설정한 경우 표시
- **설명** — 설정한 경우 2줄까지 표시

### 프로젝트 추가

우상단 **프로젝트 추가** 버튼을 클릭합니다.

| 필드 | 필수 | 설명 |
|------|:----:|------|
| 프로젝트명 | ✓ | 프로젝트 이름 |
| 상태 | | 예정 / 진행 중 / 보류 / 완료 / 취소 |
| 시작일 | | 프로젝트 시작 예정일 |
| 목표 완료일 | | 프로젝트 완료 기한 |
| 설명 | | 프로젝트 상세 내용 |

### 프로젝트에 업무 연결

업무 추가/수정 다이얼로그의 **프로젝트** 드롭다운에서 연결할 프로젝트를 선택합니다. 연결된 업무는 프로젝트 카드의 진행률 및 통계에 자동 반영됩니다.

> **참고**: 프로젝트 삭제 시 연결된 업무는 삭제되지 않지만, 프로젝트 연결이 해제됩니다.

## 8. 회신 대기함

제조사, 고객 등 **외부 응답을 기다리는 업무**만 모아서 보는 전용 뷰입니다.

### 테이블 컬럼

| 컬럼 | 설명 |
|------|------|
| 업무명 | 업무 제목 |
| 요청 대상 | 담당자 / 협력사 |
| 요청일 | 요청을 보낸 날짜 |
| 대기 일수 | 요청일로부터 경과한 일수 (색상으로 긴급도 표시) |
| 팔로업 예정일 | 재연락 예정일 (초과 시 빨간색 + `(!)` 표시) |
| 프로젝트 | 연결된 프로젝트 |
| 다음 액션 | 현재 해야 할 일 |
| 빠른 처리 | 자주 쓰는 액션 버튼 |

### 대기 일수 색상 기준

| 색상 | 기준 |
|------|------|
| 회색 | 3일 미만 |
| 기본 (테두리) | 3일 이상 |
| 빨간색 | 7일 이상 |

### 빠른 처리 버튼

각 행에서 다음 액션을 한 번에 처리할 수 있습니다.

| 버튼 | 동작 |
|------|------|
| 회신 완료 | 상태를 '진행 중'으로 변경 |
| 재문의 | 요청일을 오늘로 초기화 (재연락 기준 리셋) |
| 지연 | 상태를 '지연'으로 변경 |
| 완료 | 업무를 완료 처리 |

### 상단 통계

- **대기 N건** — 현재 회신 대기 중인 업무 총 수
- **팔로업 초과 N건** — 팔로업 예정일이 지난 업무 수 (있을 경우 빨간 배지 표시)

## 9. 설정 및 데이터 관리

### 현재 데이터 현황

설정 페이지 상단에서 저장된 데이터 통계를 확인할 수 있습니다. **새로고침** 버튼으로 최신 수치를 반영합니다.

### 샘플 데이터 불러오기

EV 충전기 관련 샘플 업무와 프로젝트를 불러와 앱의 기능을 체험할 수 있습니다.

> ⚠️ 기존 데이터가 있으면 **덮어씌워집니다**. 확인 후 진행하세요.

### JSON 내보내기 (백업)

전체 업무, 프로젝트, 활동 로그를 JSON 파일로 다운로드합니다.

- 파일명 형식: `wcb-backup-YYYY-MM-DD.json`
- 정기적으로 내보내기를 통해 데이터를 백업해 두는 것을 권장합니다.

### JSON 가져오기 (복원)

이전에 내보낸 백업 파일을 업로드해 데이터를 복원합니다.

> ⚠️ 현재 데이터를 **덮어씁니다**. 복원 후 페이지가 새로고침됩니다.

### 전체 데이터 삭제

모든 업무, 프로젝트, 활동 로그를 삭제합니다.

> ⚠️ **되돌릴 수 없습니다.** 삭제 전에 반드시 JSON 내보내기로 백업하세요.

### 데이터 저장 방식 안내

- 모든 데이터는 **브라우저 localStorage**에 저장됩니다.
- 브라우저 데이터(캐시/사이트 데이터)를 지우면 업무 데이터도 함께 삭제됩니다.
- 다른 기기나 브라우저에서 사용하려면 JSON 내보내기 후 가져오기를 사용하세요.
- **정기적인 JSON 백업을 권장합니다.**

## 10. 참조: 상태 · 우선순위 · 카테고리

### 업무 상태

| 상태 | 설명 |
|------|------|
| 신규 | 아직 착수하지 않은 새 업무 |
| 진행 중 | 현재 처리하고 있는 업무 |
| 회신 대기 | 외부(제조사, 고객 등)의 응답을 기다리는 업무 |
| 검토 필요 | 내용 확인 또는 의사결정이 필요한 업무 |
| 보류 | 잠시 중단된 업무 |
| 지연 | 예정보다 늦어지고 있는 업무 |
| 완료 | 처리가 끝난 업무 |
| 취소 | 진행하지 않기로 한 업무 |

### 업무 우선순위

| 우선순위 | 사용 기준 |
|----------|----------|
| 긴급 | 즉시 처리해야 하는 업무 |
| 높음 | 당일 또는 익일 내 처리 필요 |
| 보통 | 일반적인 처리 우선순위 |
| 낮음 | 여유 있을 때 처리해도 되는 업무 |

### 업무 카테고리

| 카테고리 | 설명 |
|----------|------|
| 장애 로그 분석 | 시스템 오류, 로그 조사 관련 업무 |
| 충전기 연동 검증 | 충전기 통신 및 연동 테스트 |
| 펌웨어 | 펌웨어 업데이트, 버그 수정 |
| 제조사 문의 | 제조사에 기술 지원 요청 |
| 고객/민원 대응 | 고객 문의 및 민원 처리 |
| 인증/시험 | 인증 취득, 시험 통과 관련 업무 |
| 프로젝트 관리 | 일정 관리, 팀 조율 등 PM 업무 |
| 내부 보고 | 보고서 작성, 내부 공유 |
| 문서 작성 | 매뉴얼, 스펙 문서 등 작성 |
| 기타 | 위 카테고리에 해당하지 않는 업무 |
